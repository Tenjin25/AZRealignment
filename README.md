# AZRealignments Mapbox GL JS Project

This project is a simple web page that displays a Mapbox GL JS map.

## How to Run

1. Open `index.html` in your browser.
2. Replace `YOUR_MAPBOX_ACCESS_TOKEN` in the HTML file with your own Mapbox token.

## Dependencies
- [Mapbox GL JS](https://docs.mapbox.com/mapbox-gl-js/guides/)

## Features
- Basic map display using Mapbox GL JS

## Customization
- Change the map style, center, or zoom in the script section of `index.html`.

---

For more advanced usage, see the [Mapbox GL JS documentation](https://docs.mapbox.com/mapbox-gl-js/guides/).
